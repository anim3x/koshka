<h4>Koshka – это мой пет-проект для изучения алгоритмов обработки данных, а также их визуального представления в веб-браузере. Цель проекта - научиться веб-скрейпингу и создать интересные с точки зрения пользователя приложения.
  
## Описание платформы
Koshka - мультиапп платформа, объединяющая в себе мини-приложения. Разработка портала ведется на Vue.js и Django.
Реализованные возможности платформы:
* Отображение данных о погоде
* Поиск кинофильмов
